Introduction

JavaScript is an object-oriented scripting language. Everything is treated as an object and can be manipulated. Think of it like this, a car is an object, and you can interact with it and change its parts, color, and etc.

JavaScript can give your website interactivity such as making a pop up or slide in menu or dropdown menu.

Knowledge Prerequisites

Before you begin writing JS code, you should:

  • You should have expert knowledge of how the World Wide Web and Internet works...Just Kidding, but a basic understanding of them is very good to have.
  • Be very familiar with HTML.
  • A little programming experience with JS or some other programming language. If you've got no experience, do visit our beginner javascript tutorial webpage and do some of the tutorials to dip your toes into the pool.
JavaScript Is Not Java

"JavaScript is a very free-form language compared to Java. You do not have to declare all variables, classes, and methods. You do not have to be concerned with whether methods are public, private, or protected, and you do not have to implement interfaces. Variables, parameters, and function return types are not explicitly typed."

Write Your 1st JS Code

Press "F12" on your keyboard to open the browser developer tools. Then click on the section (word) Console tab. Press Ctrl+L to clear the console. Click in the white space. Make sure the cursor is in the white space. Write, type this in the whitespace:

function greetMe(inputYourNameHere) { alert("Hello " + inputYourNameHere) };

Press "Enter" and then invoke the function by typing it's name followed with your name surrounded by quotes inside the parentheses:

greetMe("inputYourNameHere");

Then press "Enter."

Variables

Identifiers a.k.a variables are storing containers for data, code, objects, etc. Think of it like a box you store an item in for future use.

You can give a variable any name you want as long as it starts with a letter, underscore (_), or dollar sign ($), and numbers (0-9) can be use in the name after that, but you canNot use a hypen (-) in it. JS is case sensitive so lowercase m and uppercase M are different variables.

Examples of legal names:

  • l
  • U
  • My_Array
  • lucky7
  • _name
  • $show_me_The$
Declaring Variables

var is a keyword that is used to declare variables. Put var in front of the identifier name you want you'r variable to have. The put the assignment operator (=) after your variable's name and put number, array, object, or etc after that and a semicolon (;) at the end of the data, code, object, etc.

Number Object E.g.:

var my_made_up_variable_name = 40000000;

Object Object E.g.:

var my_object_literal = { first_name: "Johnny", "last name": "Code", age: 32 };

Array Object

var my_array_literal = ["basketball", "football", "hockey", "mixed martial arts", "soccer"];

Voila, now you have three global variables that can be accessed by anything and anywhere in your JS code, file.
